# 📚 AI Study Assistant (Pop Quiz Generator)

An AI-powered Django web app that generates interactive multiple-choice quizzes from any text using Groq’s LLaMA 3 model.

---

## 🚀 Features

- 🧠 AI-powered quiz generation using Groq API
- 📄 Convert any text into 5 MCQ questions
- 🎯 Interactive pop quiz system (one question at a time)
- 📊 Score tracking using Django sessions
- 💾 Stores quizzes using Django `JSONField`
- ⚡ Fast AI responses (LLaMA 3.1 8B Instant)
- 🌐 Ready for deployment (Render, Railway, etc.)

---

## 🛠 Tech Stack

- Python 🐍
- Django 🌐
- Groq API 🤖
- LLaMA 3.1 (8B Instant)
- SQLite (development database)
- HTML / CSS (Django templates)
